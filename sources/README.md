# Sources — 播客与访谈原始素材库

这个文件夹用于存放经典播客、YouTube 访谈的**转录文本和重点摘录**。

目的是：当知识库积累到一定规模时，做**专家观点的交叉对比与知识图谱**——看谁在同一议题上有共识，谁有分歧，分歧的背后是什么。

---

## 文件夹结构

```
sources/
├── README.md                    （本文件）
├── huberman-lab/                Andrew Huberman 的节目
├── found-my-fitness/            Rhonda Patrick 的节目
├── peter-attia/                 Peter Attia 的 The Drive
└── [其他专家]/
```

---

## 文件命名规范

每个条目一个 `.md` 文件，命名格式：

```
YYYY-MM-DD_[主题关键词].md
```

例如：
- `2026-03-30_hormones-brain-development.md`
- `2026-03-26_vitality-protocols-rhonda-patrick.md`
