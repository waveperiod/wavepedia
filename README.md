# Wavepedia

女性身体知识库。为 [Wave App](https://github.com/waveperiod) 构建。

拒绝以男性样本为默认的研究体系。

---

## 是什么

Wavepedia 收录女性荷尔蒙、代谢、神经系统、训练响应等议题的同行评审研究——以月经周期为核心坐标，给出每个阶段的具体行动指引。

每篇文章都标注：研究的受试者性别构成、荷尔蒙状态，以及对周期各阶段的实践意义。

## 文章标准

每篇文章必须包含：

1. **核心结论（一句话）** — 不用读完也知道价值
2. **机制说明** — 说清"为什么"，不只是"是什么"
3. **月经周期各阶段建议** — 卵泡期 / 排卵期 / 黄体期 / 月经期
4. **来源链接** — 原始论文或播客 URL
5. **常见误解** — 破除"以男性为默认"的错误建议

来源质量：优先同行评审研究（PubMed、PMC、Tandfonline），播客引用须追溯一手研究。
